Sound-Based Mobile Payment System.This study introduces a sound-based mobile paymentsystem as a wireless web service. Using a mobile phonefor payment for mobile commerce is essential for itssuccess. The proposed system will use sound that isgenerated from the existing mobile phone and the soundcan be recognized by the existing credit card reader of theshop with an installation of an ordinary microphone andsimple software to process sound input. This system is abetter system since it doesn't require customers to buy anew mobile phone and the cost for the microphone andthe software is very low.
